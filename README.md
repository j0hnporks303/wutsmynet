# wutsmynet

# Installation
1. Clone & place wherever
2. cd ~/wutsmynet
3. Make
4. That's it. You can now compare your network. Test your vpns, tunnels, etc etc.
# Use
1. wutsmynet
# The make file
All this is, is a very proto make file that dynamically links your file to a bin, if you don't have the bin it makes the bin for you, and then the rest comes after that. This exists but only because I felt like it, it's something I did as an experiment and because of my new developer brain forming it scratched an itch.
